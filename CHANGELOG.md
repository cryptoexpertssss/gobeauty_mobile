## [1.0.0] - 2025-01-02

### Added
- Initial release of GoBeauty mobile app
- User authentication with role-based access (client/admin)
- Beauty service booking functionality
- Browse and search beauty treatments
- View professional profiles and reviews
- Location-based salon finder with distance calculation
- Gallery view of salon services
- Beauty tips and trending services
- Appointment management for clients
- Admin dashboard for appointment oversight
- Tab-based navigation (Home, Bookings, Profile)
- Cross-platform support (iOS, Android, Web)
- Native builds via GitHub Actions
- Automatic versioning system

### Technical
- React Native 0.81.5 with Expo 54.0.20
- Expo Router 6.0.13 for file-based routing
- NativeWind 4.1.23 for styling (Tailwind CSS)
- React Query 5.83.0 for state management
- Zustand 5.0.2 for global state
- AsyncStorage for local data persistence
- TypeScript for type safety
- GitHub Actions CI/CD for automated builds
